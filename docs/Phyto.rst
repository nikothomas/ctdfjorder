Phyto (Optional Module)
========================
This module is specifically for FjordPhyto, though others may find use for it.
It has been excluded from the main package to avoid confusion.

SCAR Database
-------------
This module uses the USA, UK, Argentina, and Chile `SCAR Gazetteer Database <https://data.aad.gov.au/aadc/gaz/scar/>`__
to generate site information.
